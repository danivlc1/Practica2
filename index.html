Navegadores web

Un navegador web (en inglés, web browser) es un software, aplicación o programa que permite el acceso a la Web, interpretando distintos tipos de archivos y sitios web para que estos puedan ser vistos

La funcionalidad básiica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos incrustados.

Ejemplos de navegadores web

Safari: Es un anvegador web de código cerrado desarrollado por Apple Inc. Está disponible para macOS, iPadOS e iOS.

Mozilla Firefox: Es un navegador web libre y de código abierto desarrollado para Linux, Android, iOS y Microsoft Windows coordinado por la Corporación Mozilla.

Google Chrome: Es un navegador web de código acerrado desarrollado por Google, aunque derivado de proyectos de código abierto.
